ngFor

基础使用
  * 只能循环数组 如果需要循环对象，可以使用es6 迭代器 *
  let 变量名 of 要循环的内容
      变量名  ==>  对应的是每次循环的元素，可以直接使用插值表达式的方式进行获取
  ------------------------
  let 变量名 of 要循环的内容 [ ; 
                            [ index as i ;] 
                            [ first as 变量名 ;] 
                            [ last as 变量名 ;]
                            [ even as 变量 ;]
                            [ odd as 变量 ;]
                          ]
      index  ==>  循环位数（下标或者key）
      first  ==>  boolean 当前循环值 是否是第一个
      last   ==>  boolean 当前循环值 是否是最后一个
      even   ==>  boolean 当前循环值 是否是偶数行
      odd    ==>  boolean 当前循环值 是否是奇数行
